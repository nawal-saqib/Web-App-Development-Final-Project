The AI Movie Matrix is a full-stack movie recommendation website built using React for the frontend and Django REST Framework for the backend. It uses an SQLite 
database to store user accounts, movie details, and star ratings. The app features secure user login, smooth page routing, and clean error handling. The standout 
feature is a custom AI Recommendation Engine made with NumPy and scikit-learn. This engine looks at both a movie's genres and how other users have rated it to 
suggest personalized matches, and it includes a built-in safety backup to make sure the site never crashes even if data is missing.
